## Overview

Testers submit bugs from within a test cycle by providing details and evidence.

## Steps

1. Join a test cycle from your dashboard
2. Click "Submit Bug" in the cycle workspace
3. Fill in the bug title and description
4. Select the correct severity and environment
5. Attach screenshots or video proof
6. Submit for review

<Callout kind="info">
Moderators review and approve valid bugs.
</Callout>
